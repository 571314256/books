## 豆瓣 Top 100

### 说明
 *  这些页面是由 Python 爬虫根据图书排序算法抓取豆瓣图书信息自动生成，列出特定主题排名靠前的一百本图书。  

 *  我使用的排序算法似乎要比豆瓣默认的评价排序算法要可靠些，因为我喜欢书，尤其是对非虚构类图书有一定了解，
    所以我可以根据特定主题对排序算法进行调整。大家可以访问 [豆瓣图书爬虫](https://github.com/luozhaohui/PythonSnippet/blob/master/exportTopBooksFromDouban.py) 查看图书排序算法。
    但我并不擅长排序算法，希望能得到大家的反馈与建议，改善排序算法，提供更精准的图书排名。
